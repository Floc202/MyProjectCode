# 💝 White Valentine Website – For Bé Nấm

A small romantic interactive website created for **Bé Nấm** on **White Valentine 2026**.
The website delivers a short emotional experience with animations, music, and playful interaction.

> Created with love ❤️

---

# 🌸 Project Purpose

This project was created to celebrate **White Valentine 2026** by sending a small interactive website to Bé Nấm.

Instead of a normal message, the website provides a short romantic experience including:

* a love message
* background music
* interactive question
* playful animation

The goal is to create a cute and memorable moment.

---

# ✨ Features

* 💌 Romantic message for Bé Nấm
* 🎵 Background music that plays when the site opens
* ❤️ Floating heart visual atmosphere
* ❓ Interactive **Yes / No question**
* 🏃 “No” button that **runs away when hovered**
* 🎉 Celebration message when the user presses **Yes**

---

# 🖥️ User Experience Flow

1. The website opens with a romantic greeting.

2. Soft background music starts playing.

3. The user sees a question:

   **“Bé Nấm có thương anh không?”**

4. Two buttons appear:

   * **Có ❤️**
   * **Không 😏**

5. If the user tries to click **Không**, the button runs away.

6. When **Có** is clicked, a love message appears.

---

# 🧠 Layout Architecture

The interface uses layered visual design.

```
z-0  Background image with blur overlay
z-1  Animation layer (hearts / effects)
z-2  Main content container
z-3  Interactive buttons
```

This layering helps keep animations behind the content while maintaining a clean UI.
